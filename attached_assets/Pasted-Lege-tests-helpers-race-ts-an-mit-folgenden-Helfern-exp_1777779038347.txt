Lege tests/helpers/race.ts an mit folgenden Helfern:

  export async function runInParallel<T>(
    fns: Array<() => Promise<T>>
  ): Promise<Array<{ status: 'fulfilled'; value: T } | { status: 'rejected'; reason: unknown }>>;

  export function expectExactlyOneSuccess<T>(
    results: Awaited<ReturnType<typeof runInParallel<T>>>,
    expectedRejectionPattern?: RegExp
  ): T;

runInParallel ist ein dünner Wrapper um Promise.allSettled. expectExactlyOneSuccess prüft: genau ein 'fulfilled', alle anderen 'rejected', optional dass die Rejection-Reasons dem Pattern entsprechen — wirft sonst aussagekräftige Vitest-Assertions.
