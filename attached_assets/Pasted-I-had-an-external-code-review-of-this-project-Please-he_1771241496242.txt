## 🟠 HIGH — Verify & Fix Second

**H1 — `unhandledRejection` missing `process.exit(1)`**
Check `server/index.ts`: the `unhandledRejection` handler should call `process.exit(1)` after logging, just like the `uncaughtException` handler does. Fix if missing.

**H2 — `compression()` BREACH attack vector**
Check `server/index.ts`: `compression()` should NOT apply to `/api/` routes (risk of BREACH attack on sensitive data). Fix:
```typescript
app.use(compression({
  filter: (req, res) => !req.path.startsWith("/api")
}));
```

**H3 — Alpha package `react-signature-canvas` in production**
Check `package.json`: `react-signature-canvas` should not be an alpha version in production. This library is used for legally binding care contract signatures. Downgrade to stable `1.0.6` or replace with `signature_pad`.

**H4 — `@types/*` packages in wrong section**
Check `package.json`: `@types/bcrypt`, `@types/compression`, `@types/cookie-parser` belong in `devDependencies`, not `dependencies`. Move them.
